## 在线预览

打开 `article.html` 直接看排版效果。

## 发布到公众号

1. 登录微信公众平台 → 草稿箱 → 新建图文
2. 复制 `article.html` 全部内容粘贴到编辑器
3. 复制 `images/` 下的图片到微信素材库，在编辑器里替换 `<img src="...">` 为微信图床 url
4. 设置封面图（用 cover.png）
5. 预览 → 发布
